## Milestone Summary

**Key Decisions:**

- LIB-02 杂志式不对称网格 — **废止** (POL-04, Phase 14f)：a) auto-fill 网格在不同屏幕下视觉密度可控；b) 不对称布局对长名/短名游戏视觉权重不均衡；c) 实施成本相对收益过高。最终采用 `repeat(auto-fill, minmax(172px, 1fr))` 均匀网格
- `tauri-plugin-opener` 引入 — **Reversed** v1.2 决策 (FS-01, Phase 14a)：统一权限模型 + 平台一致 API；保留 `open_in_explorer` 兼容；新 `open_path` IPC 作为新 callsite 首选
- PER-01 跨源去重 — **查询层归并** (Phase 13a)：不动 persons 表 schema，merge_persons() 在 IPC 返回前折叠；sources / person_ids 字段暴露 attribution
- PER-04 VNDB portrait 推迟到 v1.4 (Phase 13d)：本 phase 只支持 Bangumi `/v0/persons/{id}` images.medium；VNDB-source persons 返回 None → 文字 monogram fallback
