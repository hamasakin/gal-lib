## Milestone Summary

**Decisions:**
- Independent persons + game_staff N:M tables over reflexive JSON column (locked in /gsd-explore)
- 4-role enum (scenario/artist/voice/music) — drops director/staff/producer/translator from VNDB
- 3-tier jump links (external bgm.tv/vndb.org + internal brand facet + internal /persons/:id)
- Schema once-and-done (v7 single migration; no follow-up ALTERs)

**Issues encountered:**
- VNDB rates `chardesign` and `art` as separate roles — collapsed into Artist to avoid double-row per game per artist
- Bangumi infobox `value` is sometimes string, sometimes array of {k,v} — handled with serde_json::Value coercion
- VNDB tag spoiler levels need filtering (>= 2 dropped)
- Bangumi 1req/s × 3 endpoints/game ≈ 10 min for 200-game backfill — acceptable, progress emitted via existing meta-fetch-progress channel

**Tech Debt Carried to v1.3+:**
1. Person dedup
2. Persons page enrichment (timeline / co-staff)
3. Portrait caching
4. Full backfill progress UI
