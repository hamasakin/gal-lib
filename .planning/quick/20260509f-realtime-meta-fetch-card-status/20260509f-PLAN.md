<objective>
扫描/刷新元数据时给正在 fetch 的卡片加实时高亮（脉冲边框 + spinner badge），覆盖 4 个触发场景：start_scan 首次/增量、refresh_all_metadata、refresh_metadata、bind_metadata。

Purpose: 用户在首次扫描时能立刻看到所有占位卡片出现并实时跟踪每个游戏的元数据抓取进度，提升"扫一眼就能找到游戏"的体验，同时为之后任何长耗时的元数据操作提供统一的视觉反馈语言。

Output:
- 后端拆分 ingest_one_dir 为 insert_placeholder + enrich_metadata 两阶段
- 新事件 meta-fetch-progress { game_id, phase: "started" | "finished" }
- 前端 fetchingMetaIds zustand 切片 + GameCard 高亮 + CSS pulse-ring 动画
- 启动时清理崩溃残留的孤儿占位行
</objective>
