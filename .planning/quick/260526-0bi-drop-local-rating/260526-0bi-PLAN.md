# Quick Task 260526-0bi: 移除本地用户评分字段，只保留官方评分

## 背景

当前数据库同时存在两套评分：
- `games.rating` (INTEGER 0..=10, NULL) —— 0001_init 起的本地用户打分，由 StarRating 组件维护
- `games.external_rating` (REAL 0..=10, NULL) —— v13 加的官方评分（Bangumi `rating.score` / VNDB `rating/10`）

用户决策：只保留官方评分一套，移除本地打分。

## 顺序原则（避免 v13 翻版）

**先解除引用，再 DROP COLUMN**。如果 migration 漏注册或 SELECT 比 DROP 早一步，老库 SELECT 缺列会直接 Err，触发 Library.tsx catch 块 console.error 不清空 store.games → 列表全空。

执行序：
1. 前端解除 rating 引用（Detail 页、advancedFilter、SubdirSplitDialog、StarRating 用法、i18n key）
2. 后端 SELECT/Struct/IPC 去 rating（运行时仍兼容 v13 schema）
3. **新建 v14 migration + db.rs 三处定向追加（绝对不能漏注册）**
4. cargo check + tsc + build + grep 自验
