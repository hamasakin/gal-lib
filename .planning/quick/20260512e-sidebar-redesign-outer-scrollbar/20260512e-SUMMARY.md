# Quick 20260512e — 双滚动条修复 + Sidebar 重构

## 修改文件

- `src/App.tsx` — `<main>` 加 `flex min-h-0 flex-col overflow-hidden`
- `src/components/layout/Sidebar.tsx` — 结构重设计

## 关键改动

### A. 双滚动条修复（src/App.tsx:54）

`<main>` 类名：
```diff
- <main className="min-w-0 flex-1 bg-background">
+ <main className="flex min-h-0 min-w-0 flex-1 flex-col overflow-hidden bg-background">
```

- `min-h-0` 让 flex 子项严格约束在父容器高度内 → Library 根 `h-full` 不再被子内容撑爆
- `overflow-hidden` 把滚动责任完全交给 Library 内部 `scrollContainerRef`（`overflow-auto`）
- `flex flex-col` 让 `<Outlet>` 子元素正确撑满
