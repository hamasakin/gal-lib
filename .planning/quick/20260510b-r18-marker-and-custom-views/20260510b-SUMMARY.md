## B. 自定义视图 + 批量选择

### Schema
- 同一个 v8 migration：`custom_views (id, name UNIQUE, created_at)` + `custom_view_games (view_id, game_id, added_at, PK(view_id,game_id))` + `idx_custom_view_games_game`
- 两表都 `ON DELETE CASCADE`（删视图清空 join；删游戏自动从所有视图剔除）

### 后端命令（6 个，全部注册到 `lib.rs`）
- `list_custom_views() -> Vec<CustomViewRow>` — `{ id, name, count, created_at }`，count via LEFT JOIN（空视图也返回）
- `create_custom_view(name) -> i64` — 空白 / 60+ 字符 / 重名都报中文错误
- `rename_custom_view(view_id, name)` — 同上验证
- `delete_custom_view(view_id)` — CASCADE
- `add_games_to_view(view_id, game_ids: Vec<i64>) -> i64` — 事务内 `INSERT OR IGNORE` 每个 id，返回实际新增数
- `remove_game_from_view(view_id, game_id)`
- `SearchFilter.custom_view_id: Option<i64>` + WHERE `g.id IN (SELECT game_id FROM custom_view_games WHERE view_id = ?)` 子查询
- `SidebarCategories.custom_views: Vec<CustomViewRow>` — `get_sidebar_categories` 末尾加一段 LEFT JOIN

### 前端
- `lib/customViews.ts`：6 个 wrapper + `CustomViewRow` interface
- `lib/search.ts`：`SearchFilter.custom_view_id` + `SidebarCategories.custom_views`
- `components/layout/Sidebar.tsx`：新 section "我的视图"（标题旁有 + 按钮），每行支持右键菜单（重命名 inline / 删除 confirm），无视图时显示 "还没有视图"；所有 `is*Active` checks 加 `custom_view_id == null` 子句
- `routes/Library.tsx`：批量选择模式
  - `selectMode` + `selectedIds: Set<number>` state
  - toolbar 加 "批量选择" 按钮（grid view 限定）
  - `fixed bottom-6` 浮动 bar：已选 N · 全选当前网格 · 清空 · 「添加到视图 ▾」DropdownMenu（列出现有视图 + "新建视图…"）· 取消
- `components/library/GameCard.tsx`：新 props `selectMode` / `selected` / `onToggleSelect`
  - select mode 下点击切换选中而非导航
  - 左上角原 stamp 隐藏，渲染 24px checkbox（选中态 brand 填充）
  - 选中时 cover 加 `ring-2 ring-brand` 高亮
  - 悬浮播放按钮在 select mode 下不出现
- `components/library/GameGrid.tsx`：透传三个 props
- `routes/Detail.tsx`：更多菜单加 "添加到视图" 子菜单（列视图 + 新建），单个游戏加入入口
