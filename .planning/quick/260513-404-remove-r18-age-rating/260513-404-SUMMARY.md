## Migration Strategy

- **检查:** `grep -E "^sqlx" src-tauri/Cargo.toml` → `sqlx = { version = "0.8", ... }`
- **决策:** sqlx 0.8 捆绑的 SQLite ≥ 3.42 原生支持 `ALTER TABLE ... DROP COLUMN` (3.35+)，
  直接用单条 DROP COLUMN，不走表重建保险方案。
- **Migration 0010 内容:**
  ```sql
  ALTER TABLE games DROP COLUMN age_rating;
  UPDATE app_meta SET value = '10' WHERE key = 'schema_version';
  ```
